# OpenPress 完成进度

本文档用于标识 OpenPress 当前完成情况、进行中事项和后续计划。状态会随代码推进更新。

## 状态说明

- `[x]` 已完成：仓库已有代码，并通过基础构建或类型检查。
- `[~]` 进行中：已有初版，但仍需要补强才能作为稳定能力。
- `[ ]` 未开始：尚未实现或只有规划。

## 总体进度

当前阶段：`MVP 原型可运行`

当前重点：

- 稳定 Vue 设计器交互。
- 打通模板 JSON 到打印 HTML 的真实链路。
- 补齐表格、字段、页面辅助线和属性面板基础能力。

## 已知限制

- 当前没有真实后端保存接口，playground 的“保存草稿”还只是界面入口。
- 表格分页依赖估算，不适合直接作为最终高精度票据排版能力。
- 富文本尚未做安全清洗，不应直接渲染不可信 HTML。
- 二维码和条形码还只是占位显示。
- React 设计器尚未补齐 Vue 侧的完整交互。

## 验证命令

```bash
pnpm build
pnpm typecheck
pnpm --filter @open-press/vue-playground build

```
